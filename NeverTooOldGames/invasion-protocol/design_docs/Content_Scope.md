# Invasion Protocol - Content Scope

## Levels (10 Total)

### Level 1: Tutorial
- Simple path
- 2 basic towers
- Introduces mechanics

### Level 2-3: Early Game
- Straight paths
- 3-4 towers
- Basic unit types

### Level 4-6: Mid Game
- Winding paths
- 5-6 towers
- Multiple unit types needed

### Level 7-9: Late Game
- Complex paths
- 7-8 towers
- Strategic unit combinations required

### Level 10: Final Assault
- Most complex path
- 10+ towers
- All unit types needed
- Boss challenge

## Unit Types (5-6)

1. **Grunt** - Cheap, weak, fast
2. **Tank** - Expensive, strong, slow
3. **Runner** - Fast, weak, cheap
4. **Healer** - Supports other units
5. **Siege** - High damage, slow, expensive
6. **Flying** (optional) - Ignores ground obstacles

## Progression Milestones
- Complete first level
- Unlock all unit types
- Complete all 10 levels
- 3-star all levels
- Master invasion strategies

## Replayability Features
- Different unit strategies
- Star rating challenges
- Speedrun potential
- Optimization gameplay

## Content Expansion (Future)
- Additional levels
- New unit types
- Special abilities
