# Invasion Protocol - Visual Design

## Art Style
- Dark fantasy invasion theme
- Gothic castle aesthetic
- Cartoon/stylized graphics
- Bright colors on dark background

## Color Palette
- **Background**: Dark grays, blacks
- **Castle**: Stone grays, browns
- **Player Units**: Bright reds, oranges (invaders)
- **Enemy Towers**: Blues, purples (defenders)
- **Path**: Lighter gray, clearly visible

## UI/UX Layout

### Game Screen
- Path visible (center)
- Unit selection bar (bottom)
- Currency display (top)
- Wave progress (top)
- Pause button

### Unit Selection Bar
- Icons for each unit type
- Cost displayed
- Cooldown indicators
- Hotkeys (1-6)

## Animation Requirements
- **Unit Movement**: Smooth walking animation
- **Tower Attack**: Projectile or beam
- **Unit Death**: Dramatic fall/explosion
- **Victory**: Castle breach animation
- **Currency Gain**: Floating numbers

## Visual Feedback
- **Unit Selected**: Highlight unit icon
- **Insufficient Funds**: Red flash, shake
- **Unit Spawned**: Spawn effect
- **Tower Attacking**: Visual projectile
- **Unit Damaged**: Flash red
