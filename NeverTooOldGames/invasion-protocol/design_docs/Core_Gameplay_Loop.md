# Invasion Protocol - Core Gameplay Loop

## Moment-to-Moment Gameplay
1. Earn currency passively over time
2. Select unit type to send
3. Click to spawn unit on path
4. Units auto-move toward castle
5. Enemy towers attack your units
6. Units that survive earn currency
7. Reach throne to win

## Primary Interaction
- **Unit Selection**: Choose which unit to send
- **Timing**: Send units at strategic moments
- **Resource Management**: Balance spending vs saving

## Reward/Progression Cycle

### Short-Term (Per Wave)
- Send units → Some survive → Earn currency
- Overwhelm defenses → Progress toward goal

### Medium-Term (Per Level)
- Complete level → Unlock new units
- Earn stars → Unlock harder levels

### Long-Term (All Levels)
- Complete all 10 levels
- Master all unit types
- Achieve 3-star ratings
- Unlock all content

## Engagement Loop
**Earn Currency** → **Send Units** → **Overwhelm Defenses** → **Reach Throne** → **Next Level**
