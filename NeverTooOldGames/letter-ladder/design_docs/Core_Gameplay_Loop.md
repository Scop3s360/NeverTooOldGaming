# Letter Ladder - Core Gameplay Loop

## Moment-to-Moment Gameplay
1. See start word and target word
2. Change one letter in current word
3. Submit word for validation
4. If valid, word becomes new current word
5. Repeat until reaching target word

## Primary Interaction
- **Typing**: Enter letters to modify words
- **Word Knowledge**: Think of valid intermediate words
- **Strategic Planning**: Find shortest path

## Reward/Progression Cycle

### Short-Term (Per Puzzle)
- Change letter → Valid word → Progress toward goal
- Complete puzzle → See move count
- Beat par → Earn stars

### Medium-Term (Per Session)
- Complete 10 levels
- Unlock new difficulty tiers
- Improve vocabulary

### Long-Term (Days)
- Daily challenges
- Master all levels
- Achieve 3-star ratings
- Build word knowledge

## Engagement Loop
**See Puzzle** → **Plan Path** → **Submit Words** → **Complete** → **Next Puzzle**
