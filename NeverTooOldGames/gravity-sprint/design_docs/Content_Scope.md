# Gravity Sprint - Content Scope

## Game Modes

### Endless Mode
- Infinite procedurally generated obstacles
- Progressive difficulty
- High score tracking
- Primary game mode

### Challenge Mode
- **10 Hand-Crafted Levels**
- Each level: 30-60 seconds to complete
- Specific obstacle patterns
- Time-based scoring
- Unlocked progressively

## Challenge Levels

### Beginner (Levels 1-3)
- Simple patterns
- Slow speed
- Introduction to mechanics

### Intermediate (Levels 4-6)
- Complex patterns
- Medium speed
- Combination obstacles

### Advanced (Levels 7-9)
- Very complex patterns
- Fast speed
- Precision required

### Master (Level 10)
- Ultimate challenge
- Maximum difficulty
- All obstacle types
- Bragging rights

## Unlockables
- Challenge levels unlock by reaching distance milestones in endless mode
- Level 1: Unlocked by default
- Level 2: 500m in endless
- Level 3: 1000m in endless
- Level 4: 1500m in endless
- Level 5: 2000m in endless
- Level 6: 2500m in endless
- Level 7: 3000m in endless
- Level 8: 3500m in endless
- Level 9: 4000m in endless
- Level 10: 5000m in endless

## Progression Milestones
- Reach 1000m in endless mode
- Complete all 10 challenge levels
- Achieve S-rank on all challenges
- Reach 10,000m in endless mode (mastery)

## Replayability Features
- Endless mode never ends
- Challenge mode speedrunning
- Personal best chasing
- Mastery of patterns
- Consistent difficulty curve
