# Gravity Sprint - Core Gameplay Loop

## Moment-to-Moment Gameplay
1. Character auto-runs forward at constant speed
2. Player presses button/key to flip gravity
3. Character switches between floor and ceiling
4. Avoid obstacles on both surfaces
5. Survive as long as possible

## Primary Interaction
- **Single Button**: Flip gravity (Space/Click)
- **Timing**: Press at right moment to avoid obstacles
- **Spatial Awareness**: Track obstacles on both surfaces

## Reward/Progression Cycle

### Short-Term (Per Run)
- Survive longer → Higher score
- Collect distance markers
- Beat personal best

### Medium-Term (Per Session)
- Unlock challenge levels
- Master obstacle patterns
- Improve reaction time

### Long-Term (Across Sessions)
- Complete all 10 challenge levels
- Achieve high scores
- Master advanced patterns
- Speedrun challenges

## Engagement Loop
**Run** → **Die** → **See Score** → **"One More Try"** → **Run Again**

Classic endless runner addiction loop with instant restart.
