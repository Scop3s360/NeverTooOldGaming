# Gravity Sprint - Visual Design

## Art Style
- Minimalist sci-fi aesthetic
- Clean geometric shapes
- High contrast for clarity
- Neon color accents
- Abstract, futuristic environment

## Color Palette
- **Background**: Deep space black/dark blue gradient
- **Floor/Ceiling**: Bright neon blue or cyan
- **Player**: Bright white or yellow (high visibility)
- **Obstacles**: Neon red/orange (danger colors)
- **UI**: White text with neon accents

## UI/UX Layout

### Game Screen
- Minimal HUD
- Score (top center)
- Distance traveled (optional)
- Clean, unobstructed view

### Death Screen
- Large score display
- Personal best indicator
- Quick restart button
- Return to menu button

### Challenge Select
- Grid of 10 level icons
- Locked/unlocked indicators
- Best time per level
- Difficulty indicators

- **Gravity Flip**: Screen rotation, particle trail
- **Near Miss**: Brief screen shake or flash
- **Death**: Explosion effect, screen freeze briefly
- **New High Score**: Special visual celebration
