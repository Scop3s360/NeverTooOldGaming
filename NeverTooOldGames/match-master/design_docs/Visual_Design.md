# Match Master - Visual Design

## Art Style
- Clean, polished card game aesthetic
- Theme-specific visual styles (15 unique themes)
- Smooth animations and transitions
- Professional UI with clear hierarchy

## Color Palette

### Base UI
- **Background**: Neutral tones (gray, beige)
- **Cards**: White borders with theme-specific faces
- **Buttons**: Bright, inviting colors (blue, green)
- **Text**: High contrast for readability

### Theme-Specific
Each of 15 themes has unique color palette:
1. Retro Pixel Art - Vibrant 8-bit colors
2. Minimalist Neon - Bright neons on dark background
3. Nature Botanical - Greens, browns, earth tones
4. Hand-Drawn Doodle - Sketch-style black/white with color accents
5. Dark Fantasy - Deep purples, blacks, mystical colors
6. Casino Cards - Classic red, black, gold
7. Halloween - Orange, black, purple
8. Guy Fawkes - Red, orange, yellow (fire theme)
9. Christmas - Red, green, white, gold
10. Train Railway - Industrial grays, browns, steel
11. Numbers/Math - Educational primary colors
12. Dinosaurs - Prehistoric greens, browns
13. Ocean - Blues, teals, aqua
14. Space Sci-Fi - Deep space blues, purples, stars
15. Japanese Zen - Minimalist black, white, red accents

## UI/UX Layout

### Main Menu
- Large title logo
- Play button (prominent)
- Shop button
- How to Play button
- Settings button

### Game Screen
- Card grid (centered)
- Score display (top)
- Attempts remaining (top, Medium/Hard only)
- Pause button (corner)
- Theme indicator

### Shop Screen
- Grid of theme previews
- Price tags on locked themes
- Current points display
- Purchase buttons
- Preview functionality

## Animation Requirements
- **Card Flip**: Smooth 3D rotation (0.3s)
- **Match Success**: Cards glow/pulse briefly
- **Match Fail**: Cards shake slightly before flipping back
- **Victory**: Confetti or celebration effect
- **UI Transitions**: Smooth fade in/out between screens

## Visual Feedback
- **Hover**: Card highlights slightly
- **Click**: Card depresses slightly
- **Match**: Positive visual (glow, particles)
- **Mismatch**: Negative visual (shake, red flash)
- **Purchase**: Unlock animation in shop
