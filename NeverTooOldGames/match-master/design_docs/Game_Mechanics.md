# Match Master - Game Mechanics

## Player Actions
1. **Card Selection**: Click any face-down card to flip it
2. **Memory**: Remember positions of revealed cards
3. **Matching**: Select second card to attempt a match
4. **Shop Navigation**: Browse and purchase unlockable themes
5. **Difficulty Selection**: Choose Easy, Medium, or Hard mode
6. **Pause**: Press ESC to pause game

## Rules and Constraints

### Easy Mode (4 Pairs / 8 Cards)
- Unlimited attempts
- Lower score multiplier (1x)
- Best for learning and casual play

### Medium Mode (8 Pairs / 16 Cards)
- Limited attempts (configurable)
- Medium score multiplier (2x)
- Balanced challenge

### Hard Mode (12 Pairs / 24 Cards)
- Very limited attempts (configurable)
- High score multiplier (3x)
- Maximum challenge

## Win Conditions
- Match all pairs before running out of attempts (Medium/Hard)
- Match all pairs (Easy - no attempt limit)

## Lose Conditions
- Run out of attempts before matching all pairs (Medium/Hard only)
- Easy mode has no lose condition

## Scoring System
- **Base Points per Match**: 100 points
- **Difficulty Multiplier**: 
  - Easy: 1x
  - Medium: 2x
  - Hard: 3x
- **Attempt Bonus**: Fewer attempts = higher bonus (Medium/Hard)
- **Time Bonus**: Faster completion = bonus points (optional)

## Difficulty Progression
- Players can select any difficulty at any time
- Harder difficulties offer better point rewards
- Encourages mastery and replay
