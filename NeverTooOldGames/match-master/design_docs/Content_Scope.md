# Match Master - Content Scope

## Levels/Stages
- **3 Difficulty Modes**: Easy, Medium, Hard
- **No Traditional Levels**: Procedurally shuffled card positions each game
- **15 Unlockable Themes**: Each theme is essentially new "content"

## Unlockables

### Themes (15 Total)
1. **Retro Pixel Art** - Starting theme (unlocked by default)
2. **Minimalist Neon** - 500 points

### Each Theme Includes
- 8 unique card face designs
- 1 unique card back design
- Custom background
- Theme-specific music track
- Optional: Match animation

## Progression Milestones

### Early Game (0-1,000 points)
- Learn mechanics
- Unlock first 3-4 themes
- Try different difficulties

### Mid Game (1,000-2,500 points)
- Master Medium difficulty
- Unlock 5-8 themes
- Develop strategies

### Late Game (2,500-4,000+ points)
- Master Hard difficulty
- Unlock all themes
- Achieve high scores
- Complete collection

## Replayability Features

### Variety
- 15 different visual themes
- 3 difficulty levels
- Random card shuffling each game

### Challenges (Self-Imposed)
- Complete Hard mode with minimum attempts
- Unlock all themes
- Achieve personal best scores
- Complete games with specific themes

### Progression
- Always working toward next unlock
- High score chasing
- Difficulty mastery
