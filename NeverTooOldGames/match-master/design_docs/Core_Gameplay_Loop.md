# Match Master - Core Gameplay Loop

## Moment-to-Moment Gameplay
1. Player clicks a card to flip it face-up
2. Player clicks a second card to flip it face-up
3. If cards match: Cards stay face-up, player earns points
4. If cards don't match: Cards flip back face-down after brief delay
5. Repeat until all pairs are matched

## Primary Interaction
- **Mouse Click**: Flip cards
- **Visual Memory**: Remember card positions
- **Pattern Recognition**: Identify matching pairs

## Reward/Progression Cycle

### Short-Term (Per Game)
- Match pairs → Earn points
- Complete game → Victory screen with score
- Bonus points for fewer attempts (Medium/Hard)

### Medium-Term (Per Session)
- Accumulate points across games
- Unlock new themes in shop
- Try different difficulty levels

### Long-Term (Across Sessions)
- Collect all 15 themes
- Master all difficulty levels
- Achieve high scores on each difficulty
- Complete personal challenges

## Engagement Loop
**Play Game** → **Earn Points** → **Visit Shop** → **Unlock Theme** → **Try New Theme** → **Play Game**
