## Where to get pixel art:
1. **OpenGameArt.org** - Free game assets
2. **itch.io** - Many free pixel art packs
3. **Kenney.nl** - Free game assets including pixel art
4. Create your own with **Piskel** (free online pixel art editor)

## Recommended size:
- 64x64 pixels for card faces
- Will scale up in-game as needed
