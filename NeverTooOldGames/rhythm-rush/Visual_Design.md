# Rhythm Rush - Visual Design

## Art Style
- Music visualizer aesthetic
- Pulsing, reactive UI elements
- Synthwave/retrowave inspired
- Neon colors on dark background
- Animated waveforms and particles

## Color Palette
- **Background**: Deep purple/black gradient
- **Primary**: Bright cyan/blue
- **Secondary**: Hot pink/magenta
- **Accent**: Electric yellow
- **Beat Indicators**: Pulsing white/cyan

## UI/UX Layout

### Main Game Screen
- **Center**: Large click area with beat visualizer
- **Top**: Currency display
- **Right Side**: Generator list with purchase buttons
- **Bottom**: Beat indicator bar
- **Left**: Combo multiplier display

### Generator Panel
- List of 6 generator types
- Current count owned
- Production rate
- Purchase button with cost
- Upgrade button

### Prestige Screen
- Total currency earned
- Prestige points available
- Permanent bonuses
- Prestige button

## Animation Requirements
- **Beat Pulse**: UI elements pulse with beat
- **Click Feedback**: Ripple effect on click
- **On-Beat Hit**: Bright flash + particle burst
- **Off-Beat Miss**: Subtle red flash
- **Generator Purchase**: Unlock animation
- **Combo Build**: Growing glow effect

## Visual Feedback
- **Perfect Timing**: Bright particle explosion
- **Good Timing**: Medium particle effect
- **Off-Beat**: Red flash, no particles
- **Combo Multiplier**: Growing aura around click area
- **Generator Production**: Subtle currency particles floating up
