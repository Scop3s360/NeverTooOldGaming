# Rhythm Rush - Core Gameplay Loop

## Moment-to-Moment Gameplay
1. Background music plays with clear beat
2. Visual indicators show the beat timing
3. Player clicks in rhythm with the beat
4. On-beat clicks earn currency + bonus multiplier
5. Off-beat clicks earn reduced currency
6. Use currency to buy generators
7. Generators produce passive income

## Primary Interaction
- **Rhythmic Clicking**: Time clicks to match beat
- **Strategic Purchasing**: Buy and upgrade generators
- **Combo Building**: Maintain rhythm for multiplier bonuses

## Reward/Progression Cycle

### Short-Term (Per Click)
- Click on beat → Bonus currency + multiplier
- Click off beat → Reduced currency + multiplier reset
- Build combo → Higher multiplier

### Medium-Term (Per Session)
- Accumulate currency
- Purchase generators
- Unlock new generator types
- Upgrade existing generators

### Long-Term (Across Sessions)
- Prestige system for permanent bonuses
- Unlock all generator types
- Maximize passive income
- Reach endgame milestones

## Engagement Loop
**Click to Beat** → **Earn Currency** → **Buy Generators** → **Passive Income** → **Upgrade** → **Prestige** → **Start Stronger**
