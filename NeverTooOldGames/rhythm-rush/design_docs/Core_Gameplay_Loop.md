# Rhythm Rush - Core Gameplay Loop

## Moment-to-Moment Gameplay
1. Background music plays with clear beat
2. Visual indicators show the beat timing
3. Player clicks in rhythm with the beat
4. On-beat clicks earn currency + bonus multiplier
5. Off-beat clicks earn reduced currency
6. Use currency to purchase generators

## Primary Interaction
- **Rhythmic Clicking**: Click to the beat for bonuses
- **Strategic Purchasing**: Buy generators for passive income
- **Combo Building**: Maintain rhythm for multiplier chains

## Reward/Progression Cycle

### Short-Term (Per Session)
- Click to beat → Earn currency
- Build combo → Increase multiplier
- Purchase generator → Passive income starts

### Medium-Term (Hours)
- Unlock new generators
- Upgrade existing generators
- Reach new rhythm tracks
- Build idle income empire

### Long-Term (Days)
- Prestige for permanent bonuses
- Master all rhythm tracks
- Maximize all generators
- Achieve endgame milestones

## Engagement Loop
**Click to Beat** → **Earn Currency** → **Buy Generator** → **Passive Income** → **Upgrade** → **Prestige** → **Start Stronger**
