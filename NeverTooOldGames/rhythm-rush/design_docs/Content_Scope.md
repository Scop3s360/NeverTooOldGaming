# Rhythm Rush - Content Scope

## Generator Types (6 Total)
1. **Beat Maker** - Entry level, cheap
2. **Rhythm Machine** - Early game
3. **Tempo Tower** - Mid game
4. **Melody Factory** - Late mid game
5. **Symphony Station** - Late game
6. **Harmony Hub** - Endgame

## Rhythm Tracks (5-8)
- Each track has different BPM and pattern
- Unlocked through progression
- No gameplay difference, aesthetic variety

## Upgrade Tiers
- Each generator: 10 upgrade levels
- Exponential cost increase
- Linear or exponential output increase

## Prestige System
- Unlock at 1M total currency earned
- Reset all progress
- Gain permanent multiplier bonuses
- Multiple prestige levels

## Progression Milestones
- First generator purchase
- First prestige
- All generators unlocked
- All generators max level
- Prestige level 10

## Replayability Features
- Prestige system encourages resets
- Different rhythm tracks for variety
- Optimization strategies
- Idle progression continues offline

## Content Expansion (Future)
- Additional generator types
- More rhythm tracks
- Special events
- Achievements
- Challenges (reach X currency in Y time)
