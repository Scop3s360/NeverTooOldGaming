# Rhythm Rush - Visual Design

## Art Style
- Music visualizer aesthetic
- Pulsing, reactive UI elements
- Synthwave/neon color scheme
- Clean, modern interface
- Beat-reactive animations

## Color Palette
- **Background**: Dark purple/black gradient
- **Primary**: Bright cyan/blue
- **Secondary**: Hot pink/magenta
- **Accent**: Electric yellow
- **UI**: White text with neon glow

## UI/UX Layout

### Main Screen
- Large click area (center)
- Beat indicator (visual metronome)
- Currency display (top)
- Generator list (right side)
- Upgrade buttons per generator

### Beat Indicator
- Pulsing circle that expands on beat
- Color changes: green (perfect), yellow (good), red (miss)
- Visual feedback for timing

### Generator Panel
- List of all generators
- Current count per generator
- Income per second display
- Upgrade cost and button

## Animation Requirements
- **Beat Pulse**: All UI elements pulse to beat
- **Click Feedback**: Particle burst on click
- **Perfect Hit**: Special effect for on-beat clicks
- **Combo Counter**: Growing text for combo multiplier
- **Generator Purchase**: Satisfying unlock animation

## Visual Feedback
- **On-Beat Click**: Green flash, particles
- **Off-Beat Click**: Red flash, smaller particles
- **Combo Build**: Multiplier text grows
- **Combo Break**: Multiplier fades out
- **Generator Active**: Pulsing icon
