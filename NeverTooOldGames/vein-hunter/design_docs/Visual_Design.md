# Vein Hunter - Visual Design

## Art Style
- Futuristic dwarven aesthetic
- High-tech mining equipment
- Glowing alien minerals
- Sci-fi meets fantasy
- Vibrant, colorful resources

## Color Palette
- **Background**: Deep cave blacks, dark blues
- **Minerals**: Bright glowing colors (cyan, purple, green)
- **Dwarves**: Traditional dwarf colors with tech accents
- **UI**: Metallic grays with neon highlights
- **Resources**: Rainbow of gem colors

## UI/UX Layout

### Main Screen
- Large click area (center) - the mining zone
- Currency display (top)
- Income per second (top)
- Miner list (right panel)
- Upgrade buttons per miner
- Prestige button (when available)

### Mining Zone
- Visual representation of depth
- Animated mining effects
- Resource discoveries
- Click feedback

### Miner Panel
- List of all miners
- Current count per miner
- Income per second display
- Upgrade cost and button
- Visual icons for each miner type

## Animation Requirements
- **Click Effect**: Particle burst, rock breaking
- **Miner Working**: Animated mining actions
- **Resource Discovery**: Special glow effect
- **Upgrade**: Power-up animation
- **Prestige**: Epic reset animation

## Visual Feedback
- **Click**: Satisfying particle effect
- **Purchase**: Miner appears in list
- **Upgrade**: Visual enhancement
- **Rare Resource**: Special notification
- **Prestige Available**: Glowing button
