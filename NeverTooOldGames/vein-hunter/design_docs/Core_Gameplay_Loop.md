# Vein Hunter - Core Gameplay Loop

## Moment-to-Moment Gameplay
1. Click to mine and earn currency
2. Purchase miners (generators) for passive income
3. Upgrade miners for more income
4. Unlock deeper mining levels
5. Discover rare resources
6. Prestige for permanent bonuses

## Primary Interaction
- **Clicking**: Active currency generation
- **Purchasing**: Buy and upgrade miners
- **Strategic Planning**: When to prestige

## Reward/Progression Cycle

### Short-Term (Per Session)
- Click → Earn currency
- Buy miner → Passive income starts
- Upgrade → Income increases

### Medium-Term (Hours)
- Unlock all miner types
- Reach deeper levels
- Discover rare resources
- Build idle income empire

### Long-Term (Days)
- Prestige for permanent bonuses
- Maximize all miners
- Reach endgame depth
- Achieve mastery milestones

## Engagement Loop
**Click/Mine** → **Earn Currency** → **Buy Miners** → **Passive Income** → **Upgrade** → **Prestige** → **Start Stronger**
