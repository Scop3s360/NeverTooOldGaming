# Vein Hunter - Content Scope

## Miner Types (6-8)
1. **Pickaxe Dwarf** - Entry level
2. **Drill Operator** - Early game
3. **Laser Miner** - Mid game
4. **Plasma Excavator** - Late mid game
5. **Quantum Digger** - Late game
6. **Singularity Mine** - Endgame
7. **Void Harvester** (optional) - Post-prestige
8. **Reality Breaker** (optional) - Ultimate

## Upgrade Tiers
- Each miner: 10-15 upgrade levels
- Click power: 20+ upgrade levels
- Exponential cost increase
- Exponential output increase

## Prestige System
- Unlock at 1M total currency
- Multiple prestige levels (10+)
- Each prestige: 2x permanent multiplier
- Prestige currency for special upgrades

## Mining Depths (Visual Progression)
- Surface (0-100m)
- Upper Caves (100-500m)
- Deep Mines (500-1000m)
- Crystal Caverns (1000-2000m)
- Alien Veins (2000-5000m)
- Core (5000m+)
