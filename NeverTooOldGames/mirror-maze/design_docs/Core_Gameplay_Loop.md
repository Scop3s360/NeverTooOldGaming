# Mirror Maze - Core Gameplay Loop

## Moment-to-Moment Gameplay
1. Observe level layout and goal positions
2. Plan movement considering both characters
3. Move character (shadow mirrors automatically)
4. Adjust strategy based on shadow position
5. Navigate both to their goals
6. Complete level

## Primary Interaction
- **Dual Control**: One input controls two characters
- **Spatial Reasoning**: Think about mirrored movements
- **Puzzle Solving**: Find path that works for both

## Reward/Progression Cycle

### Short-Term (Per Level)
- Plan path → Execute moves → Both reach goals
- Complete puzzle → Satisfaction

### Medium-Term (Per Session)
- Complete 10 levels
- Master mirror mechanics
- Improve move efficiency

### Long-Term (All Levels)
- Complete all puzzles
- Achieve par on all levels
- Master spatial reasoning

## Engagement Loop
**Observe Puzzle** → **Plan Path** → **Execute Moves** → **Adjust Strategy** → **Complete** → **Next Puzzle**
