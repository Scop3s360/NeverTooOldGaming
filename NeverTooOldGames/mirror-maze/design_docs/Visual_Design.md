# Mirror Maze - Visual Design

## Art Style
- Minimalist dual-world aesthetic
- Light and shadow theme
- Clean geometric design
- Clear visual distinction between characters

## Color Palette
- **Main Character**: Bright white or light blue
- **Shadow Character**: Dark gray or black
- **Background**: Neutral gray gradient
- **Platforms**: Solid colors (white/black contrast)
- **Goals**: Bright green (main), dark green (shadow)
- **Hazards**: Red

## UI/UX Layout

### Game Screen
- Single-screen puzzle (no scrolling)
- Move counter (top)
- Par display (top)
- Restart button
- Undo button (if available)

### Level Select
- Grid of 10 levels
- Move count displayed
- Par indicators
- Locked/unlocked status

## Animation Requirements
- **Character Movement**: Smooth, synchronized
- **Jump**: Both characters jump simultaneously
- **Death**: Character fades/explodes
- **Level Complete**: Both characters celebrate
- **Mirror Switch**: Visual indicator of mode change

## Visual Feedback
- **Valid Move**: Characters move smoothly
- **Invalid Move**: Characters can't move, subtle shake
- **Goal Reached**: Character glows at goal
- **Both Goals**: Level complete animation
- **Death**: Red flash, restart
