# Chroma Blend - Game Overview

## Title
**Chroma Blend**

## Tagline
"Mix Colors, Match Patterns - Learn While You Play"

## High-Level Concept
A match-3 puzzle game that teaches color theory. Instead of matching identical items, players mix primary colors to create secondary colors (red + blue = purple), then match 3+ of the same color to clear them. Educational and fun.

## Target Audience
- Casual players of all ages
- Puzzle game fans
- Educational market (art/color theory)
- Families with children

## Platform
- **Primary**: Windows PC
- **Future**: Mobile (perfect for touch)
