# Chroma Blend - Core Gameplay Loop

## Moment-to-Moment Gameplay
1. Grid filled with primary colors (red, blue, yellow)
2. Drag adjacent colors to merge them
3. Red + Blue = Purple, Red + Yellow = Orange, Blue + Yellow = Green
4. Match 3+ of same color to clear
5. New colors fall from top
6. Repeat until level objective met

## Primary Interaction
- **Drag to Merge**: Combine adjacent colors
- **Match Detection**: Create groups of 3+
- **Strategic Planning**: Plan color combinations

## Reward/Progression Cycle

### Short-Term (Per Move)
- Merge colors → Create secondary color
- Match 3+ → Clear tiles → Points

### Medium-Term (Per Level)
- Complete objective → Earn stars
- Use fewer moves → Better rating

### Long-Term (All Levels)
- Complete all 10 levels
- 3-star all levels
- Master color theory
- Learn color mixing

## Engagement Loop
**Merge Colors** → **Create Matches** → **Clear Tiles** → **Complete Objective** → **Next Level**
