# Siege and Storm - Development Phases

## MVP Features
- Path system
- Tower placement
- 2-3 tower types
- 3-4 enemy types
- 3 levels (defense only)
- Basic UI

## Phase 1: Defense Complete
- All 5-6 tower types
- All 6-8 enemy types
- 10 levels (defense phase)
- Upgrade system
- Wave spawning
- Save/load

## Phase 2: Attack Phase
- Attack phase mechanics
- Unit spawning from towers
- Enemy base
- Enemy towers (defense)
- Dual-phase gameplay

## Phase 3: Polish
- Tower animations
- Enemy animations
- Sound effects
- Background music
- UI polish
- Visual effects

## Phase 4: Balance & Features
- Balance adjustments
- Star rating system
- Level select screen
- Settings menu
- Tutorial improvements

## Phase 5: Release
- Playtesting
- Bug fixes
- Performance optimization
- Marketing assets
- itch.io/Steam release
