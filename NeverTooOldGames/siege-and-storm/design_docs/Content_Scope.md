# Siege and Storm - Content Scope

## Levels (10 Total)

### Level 1: Tutorial
- Simple straight path
- 3 waves defense
- Basic attack phase
- Introduces mechanics

### Level 2-3: Early Game
- Single path
- 5 waves defense
- Basic tower types
- Simple attack

### Level 4-6: Mid Game
- Branching paths
- 7 waves defense
- Multiple tower types needed
- Strategic attack required

### Level 7-9: Late Game
- Complex multi-path
- 10 waves defense
- All tower types useful
- Challenging attack phase

### Level 10: Final Siege
- Most complex layout
- 12+ waves defense
- Boss enemies
- Epic attack phase

## Tower Types (5-6)
1. **Arrow Tower** - Basic, balanced
2. **Cannon Tower** - Splash damage
3. **Magic Tower** - Slowing effect
4. **Barracks** - Spawns units
5. **Archer Tower** - Long range
6. **Lightning Tower** (optional) - Chain damage

## Enemy Types (6-8)
1. **Grunt** - Basic, weak
2. **Tank** - High HP, slow
3. **Runner** - Fast, low HP
4. **Flying** - Ignores ground obstacles
5. **Healer** - Supports other enemies
6. **Boss** - Very high HP, special abilities

## Progression Milestones
- Complete first level
- Unlock all tower types
- Complete all 10 levels
- 3-star all levels
- Master both phases

## Replayability Features
- Different tower strategies
- Star rating challenges
- Speedrun potential
- Optimization gameplay

## Content Expansion (Future)
- Additional levels
- New tower types
- New enemy types
- Endless mode
- Multiplayer (co-op or vs)
- Custom maps
