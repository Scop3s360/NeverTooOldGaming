# Siege and Storm - Core Gameplay Loop

## Moment-to-Moment Gameplay

### Phase 1: Defense
1. Place towers on designated spots
2. Enemies spawn and follow path
3. Towers automatically attack enemies
4. Earn currency from kills
5. Upgrade towers between waves
6. Survive all waves

### Phase 2: Attack
1. Use same towers to spawn units
2. Send units toward enemy base
3. Enemy towers defend
4. Manage resources strategically
5. Destroy enemy base to win

## Primary Interaction
- **Tower Placement**: Strategic positioning
- **Resource Management**: Balance spending and saving
- **Timing**: When to upgrade vs save

## Reward/Progression Cycle

### Short-Term (Per Wave)
- Kill enemies → Earn currency
- Upgrade towers → Stronger defense

### Medium-Term (Per Level)
- Complete defense → Unlock attack phase
- Destroy enemy base → Win level

### Long-Term (All Levels)
- Complete all 10 levels
- Unlock all tower types
- Master both phases
- Achieve 3-star ratings

## Engagement Loop
**Defend** → **Earn Currency** → **Upgrade** → **Attack** → **Win** → **Next Level**
