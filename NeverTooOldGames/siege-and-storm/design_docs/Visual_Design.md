# Siege and Storm - Visual Design

## Art Style
- Medieval fantasy cartoon graphics
- Bright, colorful aesthetic
- Stylized characters and buildings
- Clear visual hierarchy

## Color Palette
- **Player Towers**: Blue, friendly colors
- **Enemy Units**: Red, hostile colors
- **Environment**: Greens, browns (medieval)
- **UI**: Gold accents, medieval theme
- **Paths**: Lighter ground, clearly visible

## UI/UX Layout

### Game Screen
- Map (center)
- Tower selection bar (bottom)
- Currency display (top)
- Wave counter (top)
- Lives/health (top)
- Speed controls

### Tower Selection
- Icons for each tower type
- Cost displayed
- Range preview on hover
- Upgrade costs visible

## Animation Requirements
- **Tower Attack**: Projectile or effect
- **Enemy Movement**: Walking animation
- **Tower Placement**: Build animation
- **Upgrade**: Visual enhancement
- **Explosion**: Death effects
- **Victory**: Celebration animation

## Visual Feedback
- **Valid Placement**: Green highlight
- **Invalid Placement**: Red highlight
- **Tower Range**: Circle indicator
- **Enemy Damaged**: Flash red
- **Low Health**: Warning indicator
