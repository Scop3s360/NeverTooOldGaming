# Heavy Paddle - Visual Design

## Art Style
- Retro arcade aesthetic
- Modern physics visualization
- Clean, high-contrast design
- Minimalist Pong-inspired

## Color Palette
- **Background**: Black
- **Paddles**: White (with weight indicator)
- **Ball**: White
- **Center Line**: Dotted white
- **Weight Indicator**: Red (heavy) to Green (light)

## UI/UX Layout

### Game Screen
- Minimal HUD
- Score (top center)
- Weight bars above/below paddles
- Clean play area

### Weight Visualization
- Bar above paddle showing current weight
- Color gradient: Green (light) → Yellow → Red (heavy)
- Paddle visual changes with weight (thicker when heavy)

## Animation Requirements
- **Ball Movement**: Smooth, predictable
- **Paddle Movement**: Varies with weight (smooth when light, sluggish when heavy)
- **Score**: Brief flash on point scored
- **Weight Change**: Smooth bar animation

## Visual Feedback
- **Hit Ball**: Brief flash at contact point
- **Heavy Paddle**: Paddle appears thicker, darker
- **Light Paddle**: Paddle appears thinner, brighter
- **Score**: Screen flash, score increment
