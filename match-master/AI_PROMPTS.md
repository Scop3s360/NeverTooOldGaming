# AI Image Generation Prompts for Match Master

## General Tips:
- **IMPORTANT**: Add "256x256 pixels" to EVERY card prompt when generating (card faces and card backs)
- **Image Size**: Use 256x256 pixels for best quality (specify in prompt or resize after)
- Add "game asset, clean design, centered, simple" to all prompts
- Add "white background" or "transparent background" as needed
- Specify "square format" or "1:1 aspect ratio"
- For consistency, use the same prompt structure for all cards in a theme
- Generate card backs separately with "card back design, pattern"

## Recommended Image Sizes:
- **Best**: 256x256 pixels (high quality, reasonable file size)
- **Alternative**: 512x512 pixels (extra quality for high-DPI screens)
- **Minimum**: 128x128 pixels (acceptable but may look pixelated)

The game automatically scales images to fit 100x140 pixels on screen, so any size works, but 256x256 is the sweet spot.

## Tips for Best Results:

1. **Be Specific**: The more details, the better the result
2. **Consistency**: Use the same base prompt for all cards in a theme
3. **Iterate**: Generate multiple versions and pick the best
4. **Upscale**: Use AI upscalers if images are too small
5. **Edit**: Touch up in GIMP/Photoshop if needed
6. **Test**: Generate 1-2 cards first to see if style works before doing all 8

## Recommended Settings:
- **Size**: 256x256 pixels (or specify "high resolution square format")
- **Aspect Ratio**: 1:1 (square)
- **Quality**: High/Max
- **Style**: Consistent across theme
- **Background**: Specify clearly (white, black, transparent, etc.)

## Post-Processing:
After generating, you may need to:
- Remove backgrounds (remove.bg or Photoshop)
- Resize to 128x128 or 256x256 pixels
- Adjust colors/contrast
- Add transparency if needed
