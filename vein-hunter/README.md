# Vein Hunter

## Theme
Futuristic dwarven mine - dwarves with laser drills, plasma pickaxes, high-tech equipment, glowing alien minerals

## Concept
Mining-themed idle clicker where players dig deeper to find minerals, gems, and rare resources. Upgrade tools and hire miners to automate your mining operation.

## Core Mechanics
- Click to earn currency
- Buy generators that earn automatically
- Upgrade generators for more income
- Prestige system for long-term progression

## Planned Features
- Click mechanic
- Multiple generator types
- Upgrade system
- Idle income (earn while away)
- Save/load system
- Prestige/reset mechanics
