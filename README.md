# Indie Game Studio Projects

Grassroots game development - building small, salable games to fund larger projects.

## Projects

### 1. Siege and Storm
**Genre:** Tower Defense  
**Theme:** Medieval fantasy with cartoon graphics  
**Hook:** Dual-phase gameplay - defend your base with towers, then attack enemy base as towers spawn units

### 2. Vein Hunter
**Genre:** Idle Clicker  
**Theme:** Futuristic dwarven mine  
**Concept:** Dig deeper to find valuable resources, upgrade high-tech mining tools, hire dwarven miners with sci-fi equipment

### 3. Shape Shift
**Genre:** Merge Game  
**Theme:** Neon cyberpunk  
**Concept:** Merge glowing geometric shapes in a synthwave aesthetic, dark backgrounds with bright neon colors

### 4. Match Master
**Genre:** Memory Match  
**Theme:** Anime kawaii  
**Concept:** Flip cards to match cute anime-style characters and items, unlock new kawaii card sets

## Tech Stack
- Engine: Godot
- Target Platforms: Mobile (iOS/Android), itch.io, Steam

## Development Approach
- Start with minimal viable version
- Test and iterate
- Cross-platform releases
- Build audience and revenue for bigger projects
