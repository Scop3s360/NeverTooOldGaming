# Siege and Storm

## Theme
Medieval fantasy with cartoon graphics - castles, knights, goblins, bright colors, stylized art

## Concept
Dual-phase tower defense - defend against enemy waves, then attack their base using your towers to spawn units.

## Core Mechanics
- Place towers on designated spots
- Enemies follow a path
- Towers automatically attack enemies in range
- Earn currency to buy/upgrade towers
- Wave-based progression

## Planned Features
- Multiple tower types
- Enemy variety
- Upgrade system
- Multiple levels/maps
- Wave progression
